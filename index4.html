Welcome to Cakes Galore Theme

Welcome to the Cakes Galore css 3 and html 5 theme where the options are unlimited. It is a theme that supports all options of blogging, shopping and corporate portfolio. This theme does not consist of hundreds of css and js files mainly because everything on it was created from scratch. All functionality was build from scratch using all forms of coding like css 3, html 5, jquery etc.

Toggles

Discover a wide variety of toggles such as accordions and collapsible panels, tabs and much more. That includes active states as well and differen't position options.

Galleries

Multiple gallery options provided so that you can customise your site even more to the way you would want it to be instead of settling for something fixed and boring.

Sliders

View our wide range of sliders and banners that can be used any where on the site. Even for blog posts, product items and gallery items. No external plugins were used.

Icons

Explore a list of hundreds of icons from FontAwesome that can be added to any element, like your headings or just some of your buttons and form elemets.
